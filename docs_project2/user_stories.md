Below is a list of User Stories that our team worked on, split by each team member. The number at t he start of each list item is the number of story points allocated to each item.

### Shreya

* **(2)** - As a **user**, so that I can **collaborate in an existing lobby**, I want to **join using a shared invite link or code**.
* **(2)** - As a **lobby participant**, so that I can **exit gracefully**, I want to **leave the lobby at any time without affecting others**.
* **(2)** - As a **host**, so that I can **moderate interactions**, I want to **grant or revoke participant permissions** (e.g., edit, draw, speak).
* **(2)** - As a **lobby participant**, so that I can **see who’s collaborating**, I want to **view a real-time list of all connected users**.
* **(2)** - As a **developer**, so that I can **improve code maintainability and reduce technical debt**, I want to **refactor the existing lobbies code and fix related bugs**.
* **(3)** - As a **developer**, so that I can **ensure code quality and prevent future regressions**, I want to **increase the test coverage** for the lobby participation and management features.
* **(1)** - As a **developer**, so that **new team members can quickly understand the module**, I want to **create documentation** for the lobby permission and participation services.

### Shivam

* **(2)** - As a **lobby member**, so that I can **record thoughts or pseudocode**, I want to **open a shared notes editor**.
* **(2)** - As a **participant**, so that I can **collaboratively refine ideas**, I want to **edit notes and see real-time updates** from other users.
* **(2)** - As a **participant**, so that I **don’t lose work**, I want my **shared notes to be automatically saved** and persisted.
* **(5)** - As a **user**, so that I can **get automated help or hints on LeetCode problems**, I want the **system to provide an AI chatbot (Guru) backend service** that can process and respond to queries.

### Sharique

* **(2)** - As a **lobby creator**, so that I can **control who can join**, I want to **set the lobby visibility** (public or private).
